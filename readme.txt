================================
Task
================================

* Napraviti HTML tabelu koja priakzuje sliku i podatke.
* Ispod tabele treba da bude forma (bilo koji broj parametara) za unos novih podataka.	
* Forma treba da se salje preko AJAX-a i po slanju tabela treba da se azurira.
* Skripta koja obradjuje formu treba da vraca JSON u kome je i http kod.
* Podaci iz forme se unose u bazu.
* Potrebno je i napraviti dugme koje cita podatke iz tabele i pravi CSV fajl sa tim podacima. Fajl se download-uje. 
* Task je portebno uraditi bez koriscenja framework-a.

================================
Deployment notes
================================

* U fajlu bgt.sql se nalazi sql dump baze. 
* U fajlu "config.php" podesiti vrednosti za bazu (name, user, pass i host).
